# product-api

To use the API run ```go run main.go``` command in the terminal, and send requests to ```localhost:8080/products```

## Endpoints

- ```/products```, Method: **GET**
- ```/products/{id}```, Method: **GET**
- ```/products```, Method: **POST**
- ```/products{id}```, Method: **PUT**
- ```/products{id}```, Method: **DELETE**
